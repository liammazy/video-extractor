# 视频资源提取器

一个基于 Electron 的桌面应用，使用 FFmpeg/FFprobe 提取视频文件中的视频、音频、字幕资源并导出。

## 开发与运行

1. 安装依赖：

   ```bash
   npm install
   ```

2. 启动应用：

   ```bash
   npm start
   ```

选择视频文件后，界面会展示文件中的各类流。选中需要导出的资源点击“导出所选资源”，生成的文件会保存在原视频同目录，文件名前缀为原文件名。

## 说明

- 应用使用 `ffmpeg-static` 与 `ffprobe-static` 以提供内置二进制文件，无需额外安装 FFmpeg。
- 导出时使用流拷贝（`-c copy`），不会重新编码。
